Computer Science graduate and Jr. Software Engineer at **Betopia Group**, building backend services and SQL pipelines for a hybrid Web2/Web3 investment platform. I work primarily in Python, TypeScript, and SQL with a focus on clean OOP, robust schema design, and full-stack debugging. Visit my portfolio at **[radwan-ahamed.dev](https://radwanahamed.dev)** for a full overview of my work.

My projects span REST APIs, full-stack web apps, applied AI (RAG pipelines, computer vision), and blockchain backends — with production deployments on Docker, Vercel, and AWS.

Open to **Software Engineering, Backend Engineering, and Data Engineering** roles.
